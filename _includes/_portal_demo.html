Dashboard Demo

This page demonstrates the standardized dashboard structure.

URL Structure

  • All dashboard pages follow the /dashboard/* URL pattern
  • Using kebab-case for multi-word paths (e.g. /dashboard/values-assessment/)
  • Each section has its own directory with an index.html
  • All URLs end with a trailing slash

Authentication

  • All protected pages set requiresAuth: true in front matter
  • Pages are guarded by auth-guard.js
  • Using centralized authService for auth operations
  • Login page remains outside dashboard directory

Components & Includes

All dashboard pages use standardized components from the _includes directory:

_portal_account.html

_portal_exercises.html

_portal_journal.html

_portal_reflection.html

_portal_acts.html

_portal_progress.html

_portal_values_assessment.html
